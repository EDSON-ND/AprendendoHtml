O que é HTML?

(Voltar)

A sigla HTML significa Hiper Text Markup Language, isto é, linguagem de marcação de hiper textos. É um conjunto de elementos cujo objetivo é organizar a estrutura da página.

Geralmente outras tecnologias são utilizadas em conjunto na criação de uma página web, como por exemplo CSS na aparência/apresentação, e JavaScript, no funcionamento e comportamento.

HTML define o significado e a estrutura do conteúdo web. Em que pese haver menção a linguagem, não se caracteriza com uma linguagem de programação.

Alguns dizem que é o esqueleto de uma página web, fazendo uma analogia com o corpo humano.

Documento HTML

O corpo do HTML

(Voltar)

Toda estrutura de um documento HTML inicia e se encerra com uma tag denomina html. Após o início, todo código é dividido basicamente dentro de duas tags essenciais, head e body, cabeça e corpo, respectivamente, tal qual um ser humano.

No próximo tópico a temática sobre tags e suas atributos será melhor detalhado, entretanto já se pode concluir que todo documento html é estruturado em cima de tags.

< Head > - "a cabeça"

No elemento head é carregado a parte do documento HTML que não é exibida ao usuáiro quando a página é carregada. Por exemplo, ha informações sobre o título da página web, inserido através da tag title.

< Body > - "o corpo"

Nesse elemento, são carregados as informações do documento HTML que são visíveis ao usuário após a página web ser carregada.Em resumo, todos dados de textos, imagens, vídeo e interação com o usuário são desenvolvidos no corpo do HTML.

Estrutura HTML

Tags e atributos

(Voltar)

Conforme mencionado no tópicos anteriores, todo documento HTML é desenvolvido em cima de tags, que são como elementos estruturais com os quais o desenvolvedor irá organizando a página web.

As tags sempre são mencionadas no documento no interior dos símbolos maior que e menor que, < >, por exemplo, para iniciar um parágrafo iremos inserir a letra p da seguinte forma: < p >.

A maioria desses elementos possui um início e um final, ou seja, haveria uma tag de abertura e uma tag de fechamento. Esse fechamento sempre é acompanhado de uma contrabarra. No exemplo em referência o parágrafo se inicia com < p > e se encerra com < /p >.

Excepcionalmente, haverá alguns elementos que reunem essa dupla função, por exemplo, o elemento < img >, utilizado para inserir uma imagem.

Visando complementar os elementos supracitados, existem os atributos, que são palavras utlizadas no interior das tags com a finalidade de modificar, controlar, identificar uma tag.

Sintaxe HTML

Há diversas finalidade para as tags, veja abaixo alguns exemplos.

  1. Essenciais:
    • < htlm > < /htlm >;
    • < head > < /head >;
    • < body > < /body >;

  2. Títulos e subtítulos
    • < h1 > < /h1 >;
    • < h2 > < /h2 >;
    • < h3 > < /h3 >;
    • < h4 > < /h4 >;
    • < h5 > < /h5 >;
    • < h6 > < /h6 >;

  3. Formatar textos
    • < i > < /i > - itálico;
    • < strong > < /strong > - negrito;
    • < mark > < /mark > - marcador amarelo;
    • < u > < /u > sublinhado;
    • < sup > < /sup > - sobrescrito;
    • < del > < /del > - tachado;
    • < header > < /header > - cabeçalho;

  4. Inserir imagens
    • < img >;

      • Nesse elemento deve ser inserido o caminho da imagem através do atributo src = "", bem como deve se nomear a imagem através do atributo alt=""


  5. Inserir links
    • < a >;

      • Nesse elemento deve ser informado o link que será redirecionado através do atributo href="". Entre as tags deve ser informado o conteúdo que será visível.Bem como poderá ser utilizado o atributo title="" como touch tip.

      Também é possível usar o atributo target="", apontando as opções blank, para abrir o link em uma nova aba; ou self, para abrir o link na mesma aba do navegador.

      Ainda assim, mesmo optando pelo self, clicando no link segurando o -CTRL, abre-se em uma nova aba; e se clicar segurando o -SHIFT, abre-se em uma nova JANELA.


  6. Inserir listas ordenadas e não ordenadas
    • < ol > < /ol > - Listas ordenadas;
    • < ul > < /ul > - Listas não-ordenadas;
    • < li > < /li > - Listar itens;
